WiCS GitHub Workshop




GitHub Workshop!Brought to you by University of Pittsburgh Women in Computer Science Club


This website is to serve solely as a toy example for teaching club members how to contribute to 'open source projects'. Concepts covered include forking a GitHub repository, making your own contributions, and creating a pull request. This website has been co-created by WiCS club members! Thanks for participating in the workshop!





Inspiration ~ Dare Mighty Things!



Remember, when you are contributing to open source projects, you are a helper! Don't ever be afraid to fail! Contribute to open source projects to create the things you wish existed - you never know where it will take you.
